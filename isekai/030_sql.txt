価格、メニュー名を条件で並べ替え

SELECT * FROM menu ORDER BY price DESC,name;

メニューが、menu テーブルに保存されています。
menu テーブルの形式は次の通りです。

menu_id     int     メニュー番号
categoly_id int     カテゴリ番号
name        varchar メニュー名
price       int     価格

メニューを、次のように並べ替えてすべて取得してください。
・ 価格の降順に並べ替える
・ そして、価格が同じ場合は、メニュー名の昇順に並べ替える

※: メニュー名は、ひらがな、カタカナからなる文字列です。

ヒント

・ SELECT * FROM テーブル名: テーブル名 テーブルのすべてのレコードを取得することができる
・ ORDER BY カラム名1 DESC, カラム名2: 次のように並べ替えることができる
・ カラム名1 カラムの値の降順
・ そして カラム名1 カラムの値が同じとき、カラム名2 カラムの値の昇順
