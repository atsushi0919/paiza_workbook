# 注文番号、注文数を条件で並べ替え

SELECT * FROM ordered_menu ORDER BY oedered_id,num DESC;

=begin
ordered_id  int 注文番号

注文内容が、ordered_menu テーブルに保存されています。
ordered_menu テーブルの形式は次の通りです。



注文内容を、次のように並べ替えてすべて取得してください。
・ 「注文の通し番号」の昇順に並べ替える
・ そして、「注文の通し番号」が同じ場合は、「メニューの注文数」の降順に並べ替える

ヒント

・ SELECT * FROM テーブル名: テーブル名 テーブルのすべてのレコードを取得することができる
・ ORDER BY カラム名1, カラム名2 DESC: 次のように並べ替えることができる
・ カラム名1 カラムの値の昇順
・ そして カラム名1 カラムの値が同じとき、カラム名2 カラムの値の降順
